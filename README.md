Bitrix Project – Обжорозаврик🦖

Учебный проект на 1C-Bitrix.  
Проект создаётся в рамках подготовки к практике и освоения разработки на Bitrix.

🛠 Технологии
- PHP 8.1
- 1C-Bitrix
- MySQL
- Bootstrap 4
- jQuery
- JavaSCript
  

📂 Структура проекта
- `/local` — кастомный код, шаблоны, php_interface
- `/bitrix` — ядро Bitrix (не хранится в репозитории)
- `/upload` — файлы пользователей (не хранится в репозитории)

🚀 Установка и запуск
1. Установить чистый 1C-Bitrix
2. Склонировать репозиторий в корень сайта
3. Настроить подключение к базе данных
4. Проверить права на папки
5. Открыть сайт в браузере

✅ Статус проекта
- [x] Базовая структура шаблона
- [x] Подключение ассетов (Bootstrap, jQuery)
- [x] Главная страница
- [ ] Остальные страницы
- [ ] Интеграция с компонентами Bitrix

📌 Примечания
Проект не содержит ядро Bitrix и папку upload.  
Для запуска требуется установленная копия Bitrix.
